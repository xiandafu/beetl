## Beetl3 高速模板引擎

[iBeetl.com](iBeetl.com) © 2011 ~ 2019，国内流行模板引擎

#### 功能 

[http://ibeetl.com/guide/#beetl](http://ibeetl.com/guide/#beetl)

- 动态页面生成
- 静态页面生成
- 代码生成
- 通过生成XML文本中间格式间接生成PDF，WORD等格式
- 短信，微信等模板内容生成
- 脚本引擎
- 规则引擎
